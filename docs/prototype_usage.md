シフト管理プロトタイプ 起動手順
更新日: 2026-02-14

1. ターミナルでプロジェクトへ移動

```powershell
cd C:\Users\古内翔平\python_space\SHIFT_AICK_REMOTE\shift-ai-check
```

2. アプリを起動

```powershell
streamlit run prototype_streamlit_app.py
```

3. ブラウザで確認
- 自動で開かない場合は `http://localhost:8501` を開く

4. 初回の確認手順
- サイドバーで `デフォルトPOSを読込` を押す
- `POS取込プレビュー` で時間帯売上構成比を確認
- `日別全店サマリー` で指定日の全店一覧とピーク帯を確認
- `マスタ登録` で店舗/スキル/スタッフを編集して保存

5. 任意のファイル差し替え
- POS: `POS CSVアップロード` から `販売伝票明細` 形式を取込
- 予算: `日割り予算Excelアップロード` からテンプレ形式を取込

注意
- 本アプリは使用感確認用のプロトタイプです。
- 自動シフトは簡易シミュレーションで、制約最適化エンジンは未実装です。
